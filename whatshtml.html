What's HTML?

HTML stands for Hypertext Markup Language. HTML has a prolific presence on the web. It plays a huge role in web development. Here's some fun facts about HTML:
  • It was first written in 1989 by Tim Berners-Lee of CERN
  • The latest release of HTML is version 5.2
  • HTML5 has in a lot of cases replaced Adobe (Macromedia) Flash
